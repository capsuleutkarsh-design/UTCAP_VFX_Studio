# Manual Test Scripts

This directory contains manual test scripts and test outputs.

## Files

- `test_poll_worker.py` - Manual test for poll worker functionality
- `test_rbac.py` - Manual test for role-based access control
- `test_output.txt` - Test output log

## Purpose

These are manual verification scripts that complement the automated test suite. They are used for:

- Interactive testing during development
- Debugging specific components
- Validating complex workflows that are difficult to automate

## Usage

Run from project root:

```bash
python tests/manual/<script_name>.py
```
